# Flask marina backend

> Capstone project for [Bottega Code School](https://bottega.tech/)

### I have used:

- Python 3.9
- pymongo
- MongoDB + Atlas
- Jinja
- Bootstrap 5
- SCSS
- GitHub
- Postman
- Digitalocean

### Live version: 

- [Digitalocean Flask app](https://marina-back-end-wjnsy.ondigitalocean.app/)

### Open pipenv shell:

```
$ pipenv shell
```

### Run flask: 
```
$ flask run
```
